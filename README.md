# 🔐 Password-Based Digital Door Lock System
### Industrial Training Report & Presentation

Welcome to the official presentation repository for my industrial training project. This system is a practical implementation of embedded system design, utilizing an Arduino microcontroller to create a secure, password-protected locking mechanism.

## 🚀 Live Demo
You can view the interactive presentation here:  
**[Live Presentation Link](https://ritesh-1918.github.io/digital-door-lock-presentation/)**

---

## 🛠️ Technology Stack

### Hardware Architecture
- **Microcontroller**: Arduino Uno (ATmega328P)
- **Input**: 4x4 Matrix Keypad
- **Output**: 16x2 I2C LCD Display & Active Buzzer
- **Actuation**: SG90 Servo Motor
- **Power**: 5V Regulated Power Supply

### Software Stack
- **IDE**: Arduino IDE
- **Language**: Embedded C
- **Animations**: GSAP (GreenSock Animation Platform)
- **Styling**: Vanilla CSS3 with Modern Layouts (Grid/Flexbox)
